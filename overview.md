Core Concept
A comprehensive sauna wellness platform that combines:

Smart sensors (Harvia MyHarvia sensor data - temp, humidity, presence)
Voice AI coaching (ElevenLabs for hands-free interaction)
Wearable health monitoring (Apple Watch integration for heart rate, HRV)
Intelligent automation (n8n workflows for safety & personalization)

Key Features
1. Voice-Guided Sessions

AI coach guides you through traditional Finnish sauna rituals
Hands-free interaction (essential in sauna environment)
Real-time feedback based on sensor + biometric data

2. Safety Guardian

Live monitoring of heart rate + temperature
Automated alerts when thresholds exceeded
Haptic warnings on Apple Watch

3. Recovery Optimization

Post-workout sauna recommendations
Correlates exercise intensity with optimal sauna protocols
Tracks recovery metrics over time

4. Social/Community Features

See who's in Junction venue saunas
Competitive challenges
Session sharing and leaderboards

Tech Stack

Harvia GraphQL API for sensor data
n8n for workflow automation
ElevenLabs for voice interface
Apple Watch for biometrics
Supabase for database
Claude API for AI insights
